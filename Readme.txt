EX BLOG

CREATE
READ        /allBlogs
DESTROY     /updateBlog/:id
UPDATE      /destroyBlog/:id


to be able to run this code you should have the following set up
1. node.js
2. npm
3. MongoDB
4. express
5. ejs
6. npm install method-override --save
7. npm install express-sanitizer --save
8. @azure/cosmos

How to run 
node app.js open the server
go to the webpage type  localhost:3000
